# The First Blog
按照惯例，似乎应该输出一段 "Hello World！" 吧？笑。

这是我第一次接触 Jekyll，也是第一次玩 GitHub Pages。花了几个小时，从寻找中意的模板到部署 GitHub Pages（几乎是一键式操作），再到配置 _config.yml 文件。不知道为什么，每次修改 yml 文件时，总会因为一些莫名出现的空格符导致部署挫败。于是每次修改内容都变得胆战心惊，像极了小时候玩模拟器游戏时，走几步路就保存一次，生怕错过宝物或是被敌人打晕后从头再来。

第一篇博客，作为测试并不打算写太多内容。搭建这样一个博客小站是我一直以来的心愿。虽然现在的社交平台铺天盖地，但自己动手做的东西总有那么一些成就感和自由感。除了这个博客，今天还部署了一个个人网页，用于记录自己的工作经历和参与开发的游戏项目。虽然可能根本没有机会用到，但这仍然是我一直想尝试的事情。

现在的 AI 已经如此发达，导致遇到问题时，比起 Google我更习惯随手把问题或错误信息甩给 AI，让其帮忙解决，完事后还不忘亲口夸赞 AI 一番。即便如此，也有不那么顺利的时候。原本想在个人网页的基础上集成博客功能，无奈折腾半天无果，才决定放弃，索性将个人网页和博客分开，也算是曲线救国。

从事游戏开发已经快 10 年了，对于游戏之外的技术实在不太了解。就好比做一个简单的个人小站，也是折腾了大半天。还好有 AI 相助，此时此刻才得以在自己的博客里写下这些文字...去咖啡店时，比起和人挤作一团，更喜欢坐在能看到窗外风景的角落。在这个属于自己的小基地，写一点无关紧要的文字，对我来说确实格外惬意。不用因为有人阅读而歪曲本意，也无需期待被多少人读到，这些文字却始终真实存在，我喜欢这样的感觉。

家里的小猫此刻正坐在我的笔电旁边，东张西望，时不时用头蹭我，好像在提醒我该睡觉了。今天捣鼓了大半天，明天还会继续调整个人网页和博客的一些配置，有点像是在拼积木，好了今天就到这，晚安。
